    • Basic Functionality

    Books can be downloaded digitally onto Noteable.This saves students and workers from having to carry all their resources. Noteable can also act as a copy, allowing students/workers to type their work and save it as a document. Noteable also features a calculator app that allows users to make large calculations. Noteable is capable of running two applications simultaneously, by dividing the screen into two sections for both applications they can be used side by side. This feature is great if you like to take notes while reading, it saves from having to switch to the notes application.

    Additional Features and Software

    Noteable is great for sketching using its built-in e-pen. The sketch pad application allows you to use the e-pen to sketch and draw as you see fit. Your finished product can be saved in a variety of file formats such as jpeg, jpg, png and pdf. Noteable consistently allows you to save files in any file format that's compatible, throughout all of its applications. Noteable will have a camera, it will allow you to take photos of pages and convert them into pdf’s.

    Hardware

    Noteable will have an LCD screen. It will have a screen size of 7”, 600 x 1024. The tablet will have a 4GB ram that will be sufficient for running applications. It will have a battery span of 8 hours, however what happens if you run out of battery and still need to work? Noteable comes with a tablet cover that acts as a magnetic charger so you can work while charging your device without any pesky wires. The cover must be charged using a wire prior however.
