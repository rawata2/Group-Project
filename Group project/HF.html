What is it?

In the near future we can expect a huge improvement in the Display resolution, refresh rate and Texture quality of objects in games. We could get to a point very close to a level of photorealism!

Spiderman: Miles Morales

PS5's exclusive Spiderman: Miles Morales is the best looking game in PS5'S intial release. It's implementation of real-time ray-tracing make the world look real. It allows objects within the game world to create their own lighting effects, leading to more realistic looking shadows, reflections, and color shading.

Cyberpunk 2077

The most anticipated game of this century is now here. Cyperpunk 2077 delivers great graphics, real-time ray tracing, so lot of sunlights and shadows are realistic and day/night time sequences resemble real life. It has the highest graphics for the current generation but is believed to run better on next generation hardware.

Ray-tracing

the biggest improvement in the near future will most likely be real-time Ray tracing (graphics).Games today use something called Scanline rendering, which is faster than ray-tracing and necessary for high frame rates on modern hardware but shows signs of poor rendering of lighting.
